# CRUD API – Node.js & MongoDB

A simple CRUD REST API built using Node.js, Express, and MongoDB to practice backend fundamentals and API development.

### Folder Structure

```
├── 📁 src
│   ├── 📁 constants
│   │   └── 📄 constant.js
│   ├── 📁 controllers
│   │   └── 📄 product.controller.js
│   ├── 📁 db
│   │   └── 📄 db.js
│   ├── 📁 models
│   │   └── 📄 product.model.js
│   └── 📁 routes
│       └── 📄 product.route.js
├── ⚙️ .gitignore
├── ⚙️ package-lock.json
├── ⚙️ package.json
├── 📝 readme.md
└── 📄 server.js
```

### 🚀 Features

```
- Create a product
- Get all products
- Get product by ID
- Update product by ID
- Delete product by ID
```

### 🧠 Concepts Used

- Node.js & Express
- RESTful API architecture
- Express routing & controllers
- MongoDB with Mongoose
- Environment variables using dotenv
- Async/await & error handling
- Clean folder structure

### run locally

```sh
npm install
npm run dev

```
